# Astrion — Brand OS (Entrega Final)

Este repositorio contiene la implementación completa del sistema operativo de marca "Astrion Brand OS".

## 1. Instrucciones de Ejecución

Para ejecutar el proyecto localmente en su máquina:

1.  **Instalar dependencias:**
    ```bash
    npm install
    ```

2.  **Iniciar el entorno de desarrollo:**
    ```bash
    npm run dev
    ```
    Esto abrirá la aplicación en `http://localhost:5173`.

3.  **(Opcional) Construir para producción:**
    ```bash
    npm run build
    ```
